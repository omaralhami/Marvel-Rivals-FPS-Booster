## How to Use

1. Download MarvelRivalsFPSBooster.hta
2. Close Marvel Rivals game
3. Run the application
4. Click "Apply Boost"
5. Launch game and enjoy better FPS!

To restore default settings:
1. Close the game
2. Click "Remove Boost"
3. Restart the game
